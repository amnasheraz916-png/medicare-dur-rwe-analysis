# Dashboard Overview

This project contains six Tableau dashboards designed to communicate medication safety risks among elderly Medicare beneficiaries.

## Dashboard 1 – Executive Overview

Provides a high-level summary of elderly claims, beneficiaries, healthcare costs, and overall high-risk medication burden.

## Dashboard 2 – Drug Class Burden

Evaluates prescribing burden and healthcare expenditure across major high-risk medication classes.

## Dashboard 3 – Top High-Risk Drugs

Identifies medications contributing the greatest clinical and economic burden among elderly patients.

## Dashboard 4 – Provider Specialty Risk

Examines prescribing patterns across provider specialties and highlights specialty-specific medication safety risks.

## Dashboard 5 – Preventable Adverse Outcomes

Evaluates potential medication-related harms including falls, bleeding, respiratory depression, hypoglycemia, renal injury, and anticholinergic burden.

## Dashboard 6 – Deprescribing Opportunities

Identifies high-priority medications suitable for pharmacist-led medication review and deprescribing interventions.
